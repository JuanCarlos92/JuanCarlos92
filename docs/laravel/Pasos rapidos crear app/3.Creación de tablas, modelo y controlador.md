### 3º. Creación de Tabla, Modelo y Controlador

Laravel ofrece un comando muy práctico llamado **Artisan** para generar varios componentes de la aplicación de manera simultánea. En este ejemplo, crearemos un modelo para gestionar nuestras entradas de blog (posts), así como su migración, controlador, seeder y factory.

Para ello, sitúate dentro de la ruta de tu nueva App (`cd MiBlog`) y ejecuta el siguiente comando:

```bash
php artisan make:model Post -a

```

**Significado de los parámetros:**

* `-m`  Crear **migración** (estructura de la tabla).
* `-c`  Crear **controlador** (lógica).
* `-s`  Crear **seeder** (poblado de datos).
* `-f`  Crear **factory** (generador de datos aleatorios).
* `-a`  **All** (crear todos los anteriores automáticamente).

Con este único comando, agilizamos la creación de todos los componentes necesarios para manejar la entidad **Post**.
