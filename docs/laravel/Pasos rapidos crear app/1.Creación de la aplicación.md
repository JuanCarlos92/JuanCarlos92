### 1º. Creación de la Aplicación

El primer paso consiste en crear un nuevo proyecto o aplicación Laravel. Para ello, utilizamos **Composer**, que es el gestor de dependencias de PHP. Ejecuta el siguiente comando en tu terminal:

```bash
composer create-project laravel/laravel MiBlog

```

Este comando descargará e instalará Laravel en una carpeta llamada `MiBlog`.

> **Nota:** Asegúrate de tener Composer instalado previamente en tu sistema.

Una vez completado el proceso, tendrás la estructura básica de un proyecto Laravel lista para ser personalizada.
