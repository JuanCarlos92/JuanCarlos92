### 4º. Modificación de la Migración

Una vez creado el modelo, debes editar la migración correspondiente. Esta se encuentra en el directorio `database/migrations` (el archivo tendrá un nombre similar a `xxxx_xx_xx_create_posts_table.php`).

Dentro de la función `up()` encontrarás el esquema de la tabla. Agrega las columnas que definirán los atributos de cada post. Por ejemplo:

```php
Schema::create('posts', function (Blueprint $table) {
    $table->id();
    $table->string('titulo');
    $table->text('contenido')->nullable();
    $table->string('imagen')->nullable();
    $table->timestamps();
});

```

> **Dato clave:** El método `nullable()` permite que esos campos queden vacíos en la base de datos si no se proporciona información.
