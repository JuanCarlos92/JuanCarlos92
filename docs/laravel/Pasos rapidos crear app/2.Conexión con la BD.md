
### 2º. Conexión con la Base de Datos

1.  Laravel utiliza un archivo de entorno (**`.env`**) para configurar parámetros sensibles como la conexión a la base de datos. Antes de continuar, crea una base de datos en phpMyAdmin o mediante la terminal MySQL con los siguientes parámetros:
    * **Nombre:** `MiBlog`
    * **Cotejamiento (Collation):** `utf8mb4_spanish_ci`

2.  Para conectar la aplicación con la BDD, abre el archivo **`.env`** situado en la raíz de tu proyecto y modifica las siguientes líneas:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=MiBlog
DB_USERNAME=root
DB_PASSWORD='abcd1234'

```

> **Nota:** Asegúrate de que los valores de `DB_USERNAME` y `DB_PASSWORD` coincidan exactamente con los de tu servidor de base de datos local.

Esta configuración permite que Laravel se comunique con tu base de datos, lo cual es esencial para trabajar con **modelos**, **migraciones** y **seeds**.
