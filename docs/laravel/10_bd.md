## Bases de Datos en Laravel

Para trabajar con bases de datos en entornos Linux (como `/var/www/html`), es fundamental tener correctamente instalado y configurado el motor de base de datos.

---

### 8.1. Instalación de MySQL en Linux: Guía Paso a Paso

Sigue estos pasos en tu terminal para instalar y configurar MySQL.

#### 1. Actualizar la lista de paquetes
```bash
sudo apt update

```

**¿Qué hace?** Actualiza la base de datos local con la información más reciente sobre los paquetes y versiones disponibles en los repositorios.

#### 2. Instalar actualizaciones del sistema

```bash
sudo apt upgrade

```

**¿Qué hace?** Instala las versiones más recientes de los programas que ya tienes en el sistema. Si solicita confirmación, pulsa `Y` y `Enter`.

#### 3. Instalar el servidor MySQL

```bash
sudo apt install mysql-server

```

**¿Qué hace?** Descarga e instala el software de MySQL Server, encargado de gestionar las bases de datos.

#### 4. Acceder a la consola de MySQL (Modo Admin)

```bash
sudo mysql

```

**¿Qué hace?** Entra directamente al monitor de MySQL con privilegios de superusuario (`root`).

#### 5. Configurar la contraseña del usuario root

Dentro de la consola de MySQL (donde el prompt cambia a `mysql>`), ejecuta:

```sql
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '8888';

```

**Desglose del comando:**

* `ALTER USER 'root'@'localhost'`: Selecciona al administrador local.
* `IDENTIFIED WITH mysql_native_password`: Define el método de autenticación clásico (necesario para la compatibilidad con muchas aplicaciones).
* `BY '8888'`: Establece la contraseña (en este ejemplo, **8888**).

#### 6. Salir de la consola

Para volver a la terminal de Linux, escribe:

```sql
exit

```

#### 7. Verificar el acceso

Para comprobar que todo funciona correctamente, intenta loguearte con el nuevo método:

```bash
mysql -u root -p

```

**¿Qué hace?**

* `-u root`: Especifica el usuario.
* `-p`: Indica que introducirás una contraseña. Tras pulsar `Enter`, escribe `8888` (no verás los caracteres mientras escribes) y confirma.

---

> **Nota importante:** Una vez configurado MySQL, recuerda actualizar los valores `DB_USERNAME=root` y `DB_PASSWORD=8888` en el archivo `.env` de tu proyecto Laravel para establecer la conexión.
