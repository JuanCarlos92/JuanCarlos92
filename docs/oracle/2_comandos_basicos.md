## Acceder a Oracle Database

### Sin usuario y contraseña específico (SQL*Plus, con SYS como administrador):

```bash
sqlplus / as sysdba
```

### Con usuario y contraseña (la password se ingresa por pantalla):

```bash
sqlplus usuario/contraseña@XE
```

---

## Bases de datos / Esquemas

En Oracle, cada usuario funciona como un esquema. Comandos equivalentes:

### Mostrar todos los esquemas:

```sql
SELECT username FROM all_users;
```

### Crear esquema / usuario:

```sql
CREATE USER miblog IDENTIFIED BY contraseña;
GRANT CONNECT, RESOURCE TO miblog;
```

### Borrar esquema / usuario:

```sql
DROP USER miblog CASCADE;
```

---

## Tablas

### Consultar tablas del esquema actual:

```sql
SELECT table_name FROM user_tables;
```

### Describir los atributos de una tabla:

```sql
DESC nombre_tabla;
```

### Crear tabla:

```sql
CREATE TABLE empleados (
    id NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    nombre VARCHAR2(30) NOT NULL,
    edad NUMBER(3),
    salario NUMBER(10,2)
);
```

### Insertar datos:

```sql
INSERT INTO empleados (nombre, edad, salario) VALUES ('Pedro', 24, 21000);
INSERT INTO empleados (nombre, edad, salario) VALUES ('Maria', 26, 24000);
INSERT INTO empleados (nombre, edad, salario) VALUES ('Juan', 28, 25000);
INSERT INTO empleados (nombre, edad, salario) VALUES ('Luis', 35, 28000);
INSERT INTO empleados (nombre, edad, salario) VALUES ('Monica', 42, 30000);
INSERT INTO empleados (nombre, edad, salario) VALUES ('Rosa', 43, 25000);
INSERT INTO empleados (nombre, edad, salario) VALUES ('Susana', 45, 39000);
```

### Actualizar datos:

```sql
UPDATE empleados SET nombre = 'Pedro' WHERE id = 1;
```

### Obtener datos:

```sql
SELECT * FROM empleados;
```

### Borrar datos:

```sql
DELETE FROM empleados WHERE id = 1;
```

## Salir de SQL*Plus

Para salir del CLI interactivo de Oracle se usa:

```sql
EXIT;
```
